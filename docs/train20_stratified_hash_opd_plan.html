Train20 Stratified Hash OPD Plan

This document records the Section 15 active baseline matrix for the conditional OPD experiments. The active fixed schedules are uniform, late_power_3, flowts_power_sampling, ays, gits, and ots.

The active solvers are Euler, Heun / RK2, Midpoint RK2, and DPM++2M at target NFE values 4, 8, 12. Metrics are CRPS and MASE.

The training protocol uses a single scalar MLP teacher on 20% train examples chosen by temporal stratified hash sampling, followed by validation_tuning checks for the older V4 selection path. Teacher rows use behavior-balanced weighting across the active schedule behaviors.
